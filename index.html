Core Idea

A neural network adjusts weights and biases to make the model fit the data better. Training is just finding the right values for these two things.

Weight vs Bias

Weight — controls how much each input matters (slope)
Bias — controls the activation threshold (intercept)

  • Neuron — takes inputs, multiplies by weights, adds bias, passes through activation function
  • Activation function — adds non-linearity so the network can learn complex patterns, not just straight lines
  • Training goal — find weights & biases that minimize prediction error
 What do weights and biases do in a neural network?
Weights scale each input signal to reflect its importance, and biases shift the activation threshold. Training is the process of finding the weight and bias values that minimize prediction error across the training data.
In my LSTM model for neonatal pain detection, the weights learned how much each physiological signal (heart rate, facial AU) contributes to the pain prediction at each timestep. The bias terms helped calibrate the activation threshold for each neuron in the hidden layers.
